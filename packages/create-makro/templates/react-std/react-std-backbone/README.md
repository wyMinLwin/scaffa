# React Standard Template

## Tech Stack

**Client UI:** React, Typescript, TailwindCSS, Shadcn

**Client Data Fetch:** Axios, Tanstack

## Run Locally

**Install Dependencies**

```sh
pnpm install
```

**Run Local Server**

```sh
pnpm run
```
